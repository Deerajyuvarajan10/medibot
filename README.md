# 🩺 Medibot AI

An AI-powered medical chatbot web application that helps users interact with an intelligent healthcare assistant using natural language.

## 🚀 Features

- 🤖 AI-powered chatbot
- 💬 Real-time medical conversations
- 🌐 Full-stack MERN architecture
- 🔥 Firebase frontend hosting
- ☁️ Render backend deployment
- 🧠 Gemini API integration
- 📱 Responsive modern UI
- 🔒 Secure environment variable handling

---

# 🛠️ Tech Stack

## Frontend
- React.js

- CSS / Tailwind (if used)
- Firebase Hosting

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/medibot.git
cd medibot
```

---

# 🔧 Backend Setup

## Move to Backend

```bash
cd backend
```

## Install Dependencies

```bash
npm install
```

## Create `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_url
GEMINI_API_KEY=your_api_key
```

## Start Backend

```bash
npm run dev
```

or

```bash
node server.js
```

---

# 💻 Frontend Setup

## Move to Frontend

```bash
cd frontend
```

## Install Dependencies

```bash
npm install
```

## Create `.env`

```env
VITE_BACKEND_URL=http://localhost:5000
```

## Run Frontend

```bash
npm run dev
```
